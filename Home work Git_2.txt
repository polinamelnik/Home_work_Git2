1. Открыть терминал и перейти в рабочий каталог проект. Следующий шаг это - инициализация локольного Git репозитория с помощью команды "git init".

1.1. Далее нужно добавить файлы в проект с командой "git add ." , "." добавит все файлы в текущем каталоге в индекс. Чтобы добавить только определенные файлы, можно указать их имена вместо ".". 

1.2. Следующим этапом я создаю коммит с командой "git commit -m "Initial commit" ".
(Initial commit - заменить на комментарий).

1.3. Если нет репозитория на GitHub, то следует его создать, и связать его с локальным репозиторием на GitHub через команду "git remote add origin <URL_удаленного_репозитория>" (обезательно заменить на личный удаленный репозиторий).

1.4. Через команду "git push -u origin master" - отправить локальный репозиторий на GitHub. 
(В этом примере используется ветка `master`. Если используется другая ветка, ее нужно заменить на имя вашей ветки).


2.  Чтобы отказаться от изменений в файле (файлах) в рабочем каталоге, следует использовать команду: git checkout -- <название_файла>. (Чтобы отменить изменения во всех файлах можно использовать ".". 

3. Для того чтобы вернуть файл из индекса,используется команда "git reset HEAD <имя_файла>" . Также можно использовать "." , для того чтобы вернуть все файлы.

4. Чтобы отменить ошибочный commit, следует использовать команду git revert.
(< > в угловых скобках написать идентификатор коммита который нужно отменить.Команда создаст новый коммит, который отменяет изменения указанного коммита.
4.1 Также можно использовать команду "git reset ", но с осторожностью т.к.  (!!!! эта команда может создать проблемы синхронизации истории коммитов!!!!!!!!!)
